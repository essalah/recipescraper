To scrap recipes from allrecipes.com

  • Call this link below by passing a recipe url in 'url parameter' see the example below :

  • * Allrecipes.com scraper https://test1320.herokuapp.com/fromurl=[pass the recipe url here and go!!]
  • http://localhost:8080/fromurl?url=https://www.allrecipes.com/recipe/261303/julis-fool-proof-rib-roast
